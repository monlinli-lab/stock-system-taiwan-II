# 台股兵法智謀全媒體觀測系統

這是已整理成 **GitHub 可管理的 Vite + React + TypeScript 專案**。

## 本機執行
```bash
npm install
npm run dev
```

## 建置
```bash
npm run build
```

## 上傳 GitHub
1. 建立新的 GitHub Repository
2. 將整個專案資料夾內容上傳
3. 若要部署到 Vercel / Netlify，可直接匯入此 Repository
4. 若要部署到 GitHub Pages，建議再加上 Pages 部署流程或改用 Vercel 會更簡單

## 專案結構
- `src/App.tsx`：主畫面
- `src/main.tsx`：React 入口
- `src/index.css`：基本樣式
- `index.html`：Vite 首頁
- `package.json`：套件與指令
